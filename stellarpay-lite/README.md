# StellarPay Lite

StellarPay Lite is a beginner-friendly Stellar Testnet decentralized application built with HTML5, CSS3, Vanilla JavaScript, Node.js, and npm.

## Project Structure

- assets/ — static images and icons
- css/ — stylesheet
- js/ — JavaScript modules
- screenshots/ — preview screenshots

## Getting Started

1. Open the project folder.
2. Run the local server:
   ```bash
   npm start
   ```
3. Visit http://localhost:3000 in your browser.

## Notes

This starter project only sets up the initial structure and placeholder files. Functionality will be added in future steps.
